# AWS Portfolio Website

## Project Overview
This project is a personal portfolio website built using HTML, CSS, and JavaScript.
The website is hosted on Amazon S3 and delivered globally using Amazon CloudFront.

## Technologies Used
- HTML
- CSS
- JavaScript
- Amazon S3 (Static Website Hosting)
- Amazon CloudFront
- AWS IAM

## Website Sections
- About Me
- Skills
- Projects
- Education
- Contact

## Deployment Steps
1. Created a responsive portfolio website using HTML, CSS, and JavaScript.
2. Uploaded website files to GitHub repository.
3. Created an S3 bucket and enabled static website hosting.
4. Configured bucket policy for public access.
5. Created CloudFront distribution for global content delivery.
6. Verified website using CloudFront URL.
